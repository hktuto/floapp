# 功能規格

## 1. 帳戶系統 (accounts + user_profiles)

### 頁面
- `/login` - 登入頁面
- `/register` - 註冊頁面（獨立頁面）

### 註冊/登入流程
1. **新用戶註冊**: `/register` → `/welcome` → `/record-first` → `/first-target` → `/dashboard`
2. **現有用戶登入**: `/login` → `/dashboard` (已有資料) 或 `/welcome` (資料不完整)
3. **Auth Middleware**: 未登入訪問 private route 自動跳轉至 `/login`

### Public Routes（無需登入）
- `/` - 首頁
- `/login` - 登入
- `/register` - 註冊

### Private Routes（需登入）
- `/dashboard` - 儀表板
- `/welcome` - 歡迎/填寫個人資料
- `/record-first` - 第一個記錄
- `/first-target` - 第一個目標
- `/result` - 記錄結果比較
- `/record` - 記錄頁面
- `/friends` - 朋友列表
- `/settings` - 設定

---

## 2. Onboarding 流程（新用戶引導）

### `/welcome` - 歡迎頁面
**Layout**: `onboarding`（無底部導航）

**內容:**
- Step Indicator (1️⃣-2-3)
- 歡迎訊息 + 慶祝圖標
- 個人資料表單：名/姓（必填）、性別、身高、出生日期

**行為:** 保存後跳轉到 `/record-first`

---

### `/record-first` - 第一個記錄
**Layout**: `onboarding`

**內容:**
- Step Indicator (✅-2️⃣-3)
- 體重輸入（必填）
- 「更多數據」折疊區域
- 下一步按鈕

**行為:** 保存後跳轉到 `/first-target`

---

### `/first-target` - 第一個目標
**Layout**: `onboarding`

**內容:**
- Step Indicator (✅-✅-3️⃣)
- 目標類型選擇
- 目標數值 + 日期
- **Skip 按鈕** - 可以跳過

**行為:** 設定目標或 Skip → `/dashboard`

---

## 3. Dashboard 儀表板

### 新增記錄 Drawer
- **觸發**: 右上角「記錄」按鈕
- **組件**: `UDrawer` with `scale-background`
- **內容**: `MeasurementForm` 組件
  - Tabs 分類：「身體組成」+「身體圍度」
  - 使用 SlideInput 組件

### 保存後行為
```
有舊記錄？→ 跳轉到 /result (顯示變化比較)
冇舊記錄？→ 刷新 Dashboard
```

---

## 4. Result 結果頁面

**路徑**: `/result`
**Layout**: `onboarding`

### 用途
用戶保存新記錄後，如果有舊記錄，自動跳轉到呢個頁面顯示變化同目標進度。

### 內容區塊
1. **成功提示** - 慶祝動畫 + 記錄成功
2. **最新記錄** - 體重、體脂等數據
3. **舊記錄比較** - 同上次比較（綠色=進步，紅色=退步）
4. **第一個記錄提示** - 如果冇舊記錄
5. **目標進度** - 進度條 + 百分比
6. **操作按鈕** - 「再記一筆」或「完成」

## 5. Auth 系統

### useAuth Composable
```typescript
const { 
  user, account, isLoggedIn, isLoading, error,
  login, register, logout, fetchUser, checkAuth
} = useAuth()
```

### Auth Middleware
- 自動保護所有 private 頁面
- 未登入自動跳轉 `/login`

## 6. 技術注意事項

### Database Insert 修復
由於 Drizzle ORM + PGlite 對 `decimal` 類型處理有問題，改用原生 PGlite client：
```typescript
const client = (db as any).$client
const result = await client.query(sql, params)
```
