# Flo App Documentation

## 簡介
Flo App 係一個體重同體脂記錄網頁應用，正擴展為運動課程預約平台。

## 當前版本 (v1.0)
- 體重/體脂記錄
- 進度追蹤
- 朋友互動
- 目標設置
- Auth 系統（Login/Register）
- Onboarding 流程

## 擴展功能 (v2.0 規劃中)
- Avatar 頭像支持
- 角色系統（Admin/Teacher/User）
- Token 代幣經濟
- 運動課程預約系統

## 技術棧
- Nuxt 4
- Nuxt UI v4
- Nuxt Hub (PGlite + Blob)
- 用戶名/密碼認證
