# Flo App Documentation

## 簡介
Flo App 係一個體重同體脂記錄網頁應用。

## 功能
- 體重/體脂記錄
- 進度追蹤
- 朋友互動
- 目標設置

## 技術棧
- Nuxt 4
- Nuxt UI v4
- Nuxt Hub (PGlite + Blob)
- 用戶名/密碼認證

## Pages
- `/` - 首頁
- `/login` - 登入/註冊
- `/record` - 記錄體重
- `/friends` - 朋友列表
- `/settings` - 個人設置
