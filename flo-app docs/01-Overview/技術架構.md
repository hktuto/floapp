# 技術架構

## 架構圖

```
┌─────────────────┐
│   Frontend      │  Nuxt 4 + Nuxt UI v4
│   (SSR/SPA)     │
└────────┬────────┘
         │
┌────────▼────────┐
│   Nuxt Hub      │  Edge Functions
│   (Server)      │
└────────┬────────┘
         │
    ┌────┴────┐
    ▼         ▼
┌────────┐ ┌────────┐
│ PGlite │ │  Blob  │
│  (DB)  │ │Storage │
└────────┘ └────────┘
```

## 技術選型理由

### Nuxt 4
- Full-stack Vue framework
- 自動路由
- Server API

### Nuxt UI v4
- Tailwind CSS based
- 完整組件庫
- 深色模式支持

### Nuxt Hub + PGlite
- Edge-ready
- PostgreSQL compatible
- 無需外部數據庫服務器

### 認證方式
- 支持多種登入類型：username / email / OAuth
- 使用 `identifier` + `hash` 模式
- 未來可擴展 Google / Apple OAuth
- 所有 ID 使用 UUID v7

### 認證中間件 (auth.global.ts)
- Global middleware：自動保護所有頁面
- Public routes：`/`、`/login`、`/register` 無需登入
- 其他頁面：未登入自動跳轉至 `/login`
- 使用 cookie (`session`) 檢查登入狀態
