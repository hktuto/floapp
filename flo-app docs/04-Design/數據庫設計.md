# 數據庫設計

## 說明
- 所有 ID 使用 UUID v7（後台自建，非 auto increment）
- 時間戳統一使用 TIMESTAMP with time zone
- JSONB 用於需要靈活擴展的字段

---

## 表格結構

### accounts
用戶帳戶表，支持多種登入方式

| 字段 | 類型 | 說明 |
|------|------|------|
| id | UUID PK | 帳戶ID (UUID v7) |
| type | TEXT | 登入類型: `username` / `email` / `oauth` / `google` / `apple` |
| identifier | TEXT UNIQUE | 登入識別碼（username / email / OAuth ID）|
| hash | TEXT | 密碼哈希 / OAuth token hash |
| meta | JSONB | 擴展數據（OAuth provider info 等）|
| is_active | BOOLEAN DEFAULT true | 是否啟用 |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 創建時間 |

**索引:**
- `identifier` (UNIQUE)
- `type + identifier` (複合索引)

---

### user_profiles
用戶個人資料

| 字段 | 類型 | 說明 |
|------|------|------|
| account_id | UUID PK FK | 帳戶ID |
| first_name | TEXT | 名 |
| last_name | TEXT | 姓 |
| gender | TEXT | 性別: `male` / `female` / `other` |
| height_cm | DECIMAL(5,2) | 身高 (cm) |
| birth_date | DATE | 出生日期 |
| latest_record | JSONB | 嵌入最新記錄（緩存）|
| preferences | JSONB | 用戶偏好設置（語言、主題、提醒等）|
| created_at | TIMESTAMPTZ DEFAULT NOW() | 創建時間 |
| updated_at | TIMESTAMPTZ DEFAULT NOW() | 更新時間 |

**索引:**
- `account_id` (PK)

---

### targets
目標設定

| 字段 | 類型 | 說明 |
|------|------|------|
| id | UUID PK | 目標ID (UUID v7) |
| account_id | UUID FK | 帳戶ID |
| target_type | TEXT | 類型: `weight` / `body_fat` / `muscle` / `waist` / `chest` 等 |
| target_value | DECIMAL(6,2) | 目標數值 |
| target_date | DATE | 目標日期 (可選) |
| status | TEXT DEFAULT 'active' | 狀態: `active` / `completed` / `abandoned` |
| achieved_at | TIMESTAMPTZ | **達成日期** |
| is_active | BOOLEAN DEFAULT true | 是否啟用 |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 創建時間 |

**索引:**
- `account_id`
- `account_id + status` (複合索引)

---

### measurement_records
測量記錄（體重、體脂、圍度等）

| 字段 | 類型 | 說明 |
|------|------|------|
| id | UUID PK | 記錄ID (UUID v7) |
| account_id | UUID FK | 帳戶ID |
| recorded_at | DATE | 記錄日期 |
| weight_kg | DECIMAL(5,2) | 體重 (kg) |
| body_fat_pct | DECIMAL(4,2) | 體脂 (%) |
| muscle_kg | DECIMAL(5,2) | 肌肉量 (kg) |
| water_pct | DECIMAL(4,2) | 身體水分 (%) |
| bone_kg | DECIMAL(4,2) | 骨骼密度 (kg) |
| visceral_fat | INTEGER | 內臟脂肪 (1-59) |
| chest_cm | DECIMAL(5,2) | 胸圍 (cm) |
| waist_cm | DECIMAL(5,2) | 腰圍 (cm) |
| hips_cm | DECIMAL(5,2) | 臀圍 (cm) |
| thigh_cm | DECIMAL(5,2) | 大腿圍 (cm) |
| arm_cm | DECIMAL(5,2) | 手臂圍 (cm) |
| note | TEXT | 備註 |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 創建時間 |

**索引:**
- `account_id`
- `account_id + recorded_at` (複合索引，用於查詢歷史)

---

### attachments
附件（照片、文件）

| 字段 | 類型 | 說明 |
|------|------|------|
| id | UUID PK | 附件ID (UUID v7) |
| account_id | UUID FK | 帳戶ID |
| record_id | UUID FK | 關聯記錄ID (measurement_records.id) |
| file_name | TEXT | 檔案名稱 |
| content_type | TEXT | MIME type (e.g., `image/jpeg`) |
| blob_path | TEXT | Blob storage path |
| file_size | INTEGER | 檔案大小 (bytes) |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 創建時間 |

**索引:**
- `account_id`
- `record_id`

---

### friendships
朋友關係

| 字段 | 類型 | 說明 |
|------|------|------|
| id | UUID PK | ID (UUID v7) |
| requester_id | UUID FK | 發送請求者 (accounts.id) |
| addressee_id | UUID FK | 接收請求者 (accounts.id) |
| status | TEXT DEFAULT 'pending' | 狀態: `pending` / `accepted` / `rejected` |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 創建時間 |

**索引:**
- `requester_id`
- `addressee_id`
- `requester_id + addressee_id` (UNIQUE，防止重複請求)

## ER 圖

```
accounts ||--o{ user_profiles : has
accounts ||--o{ measurement_records : has
accounts ||--o{ targets : has
accounts ||--o{ attachments : has
accounts ||--o{ friendships : requests
accounts ||--o{ friendships : receives
measurement_records ||--o{ attachments : has
```

---

## 注意事項

1. **UUID v7**: 所有 ID 由應用層生成，使用 UUID v7（時間排序）
2. **latest_record**: 用戶最新記錄的快照，更新 measurement_records 時同步更新
3. **preferences**: 存儲用戶 UI 偏好（主題、語言、提醒設置等）
4. **achieved_at**: 目標達成時設置，而非歸檔時間
